# VS Code

## VS Code

`brew cask install visual-studio-code`

[VS Code](https://code.visualstudio.com/)，微软家的开源代码编辑器  
其他主流的 GUI 编辑器还有 WebStorm、Sublime Text 等，  
但 VS Code 绝对是主流之一。

### VS Code 基本功能

- 插件市场
- 内建 git 管理功能
- snippets
- settings
- 代码格式化
- 快捷键系统
  - `cmd + p`、`cmd + shift + p`
  - `cmd + f`、`cmd + shift + f`
  - `cmd + d`、`cmd + shift + L`
  - `opt + shift + o`：organize imports
- terminal
- diff 功能
